Ready2Go
========

Ready2Go is a Rasbian (Debian) based distro for the RasberryPi

Base OS : Rasbian  

How do I install this ?

wget https://raw.github.com/arjitc/Ready2Go/master/install.sh

chmod +x install.sh

./install.sh
